The Documentation SupplyChain on TAP

Dokyung published on 2023-03-24 included in Documentation
TANZU SUPPLYCHAIN 아래 내용은 기본적으로 VMWARE의 TAP솔루션을 알고 있다는 가정하에 작성을 하였습니다. CI/CD는 (Continuous Integration/Continuous Delivery) 지속적인 통합 및 지속적인 배포입니다. 예를 들어: 사용자는 IDE 환경 (IntelliJ or visual studio or eclipse)에서 소스코드를 수정 및 디버깅을 합니다. 이 후에 수정된 소스코드를 GIT에 PUSH를 합니다. JENKINS에서는 GIT에 수정된 소스를 Webhook 또는 Polling을 통해 변경을 감지 합니다. JENKINS에서 수정된 코드를 통해 Docker Image를 생성을 합니다. 생성된 Docker Image를 이미지 레포지토리에 업로드 합니다.
